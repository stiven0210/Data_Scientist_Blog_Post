Project: Writing a Data Scientist Blog Post

First Udacity project, according to a data delivered, several questions must be determined when applying data science

Every year, Stack Overflow conducts a massive survey of people on the site, covering all sorts of information like programming languages, salary, code style and various other information. This year, they amassed more than 64,000 responses fielded from 213 countries.

The data is made up of two files:

  1. survey_results_public.csv - CSV file with main survey results, one respondent per row and one column per answer
  2. survey_results_schema.csv - CSV file with survey schema, i.e., the questions that correspond to each column name

Answer: Most people clearly start to work arround 2:00 PM.

Do the people start to work earlier in USA or India?
Answer: People in India start work at 07:00 PM and people in the United States at 01:00 PM

Do people who work from home start work earlier?
Answer: It seems that no one disagrees not to work from home, the graph shows that more work is done in the morning around 10:00 AM
